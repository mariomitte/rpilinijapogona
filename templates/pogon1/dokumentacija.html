PROJEKT LINIJA POGONA

Osnovne cjeline od kojih se sastoji projekt su mbed MCU razvojna pločica, RPi model 3 i Arduino. Projekt ostvaruje upravljanje lokalno ili na daljinu sa sušarom, lancima. Središnji dio projekta je RPi model 3 na kojemu je postavljen Django web okruženje(web poslužitelj) i upravljačka stranica pomoću koje se može ručno upravljati linijom pogona. Linijom se upravlja pritiskom na nekoliko tipki na upravljačkoj stranicic a čija je interakcija popraćena sa statusnim signalima na upravljačkoj stranici.

Tipkala kreiraju "Http GET request" zbog kojih server odgovara upravljačkim signalima sa izvršnim elementima linije pogona.

Tipkalima su dodijeljene prikladne JavaScript funkcije:

Kamera: stop_record() | record() | take_photo()

Aktuator: fwd() | bwd() | lijevo() | desno() | pauziraj() | stop_linija()

UPRAVLJAČKA PLOČA

Kreirana upravljačka ploča za pogon1 i upravljanje vagonima, sušarom, lancima

Django web okruženje zasniva se na uređivanju metoda u svega tri osnovna modela Models.py, Views.py i Template.

->Model Models.py: kreira nove modele za pohranu u bazi podataka

->Model Views.py: Obrađuje zahtjeve klijenta i prikazuje Template

->Model Template: Za izradu stranice upravljačka ploča

Django web okruženje obrađuje zahtjev prema slijedećoj pojednostavljenoj shemi:

{% svg 'DJANGOMODEL' %}

Tipkalima su pridružene funkcije za upravljanje. Pritiskom na tipkala aktivira se GET zahtjev u smjeru klijent->server, oblika "?cmd=stop"

STRUKTURA DATOTEKA - RPi3

{% svg 'STRUKTURADATOTEKA' %}

Osnovna struktura sastoji se od dijela koji pripada klijentu i dijela koji pripada serveru

Značajne su slijedeće datoteke: config - u kojoj se uređuje vrsta baze podataka, lokacija Template datoteke, URL mapiranja, itd., i datoteka pogon - aplikacija za liniju, tu je još i datoteka api - implementacija Django REST Framework-a

Python izvršne skripte camera_network.py i mbedI2C.py imaju ulogu implementiranja rada sa kamera modulom i RPI.GPIO pinovima.

ESP8266 serializer - za dohvaćanje tokena

Ovime se ostvaruje upravljanje aktuatorima preko ESP8266 modula u jednom smjeru.

ESP modul u jednakim vremenskim intervalima dohvaća status linije pogona za što je kreiran Django REST okruženje - API za dohvaćanje statusa linije pogona pomoću drugih servisa kao što je Telegram
